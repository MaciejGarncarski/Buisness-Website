
# Business website
A website that represents some not existing company. Made for learning purposes :smile:.

## Tech/framework used 🔧
- React
- StyledComponents
- Typescript
- Framer Motion
- React Leaflet
- Swiper.js

## Instalation & running 💾

### Install node dependencies
```
yarn install
```

### Start dev server
```
yarn dev
```
